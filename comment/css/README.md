# CSS Comments Snippets

### [cc-basic] basic

```css
/* ${1:Basic comment} */
```

### [cc-block] block

```css
/**
 * ${1:Block comment}
 *
 */
```

### [cc-section] section

```css
/* ==========================================================================
   ${1:Section}
   ========================================================================== */
```

### [cc-sub-section] sub-section

```css
/* ${1:Sub-section}
   ========================================================================== */
```
