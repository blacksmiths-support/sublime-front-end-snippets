# CSS Comments Snippets

Prefix `cc*`

### [ccbasic] basic

```css
/* ${1:Basic comment} */
```

### [ccblock] block

```css
/**
 * ${1:Block comment}
 *
 */
```

### [ccsection] section

```css
/* ==========================================================================
   ${1:Section}
   ========================================================================== */
```

### [ccsubsection] sub-section

```css
/* ${1:Sub-section}
   ========================================================================== */
```
