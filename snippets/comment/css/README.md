# CSS Comments Snippets

Prefix `xc*`

### [xcbasic] basic

```css
/* ${1:Basic comment} */
```

### [xcblock] block

```css
/**
 * ${1:Block comment}
 *
 */
```

### [xcsection] section

```css
/* ==========================================================================
   ${1:Section}
   ========================================================================== */
```

### [xcsubsection] sub-section

```css
/* ${1:Sub-section}
   ========================================================================== */
```
