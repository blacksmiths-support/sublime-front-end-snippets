# HTML Comments Snippets

Prefix `xh*`

### [xhbasic] basic

```html
<!-- ${1:div} -->
```

### [xhblock] block

```html
<!-- ${1:title}
${2:description}
-->
```

### [xhcond] conditional

```html
<!--[if ${1:IE} ${2:8}]>
  ${3}
<![endif]-->
```

### [xhend] end tag

```html
<!-- // ${1:div} -->
```
