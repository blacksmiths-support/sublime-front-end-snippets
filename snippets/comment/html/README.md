# HTML Comments Snippets

Prefix `hc.*`

### [hc.basic] basic

```html
<!-- ${1:div} -->
```

### [hc.block] block

```html
<!-- ${1:title}
${2:description}
-->
```

### [hc.cond] conditional

```html
<!--[if ${1:IE} ${2:8}]>
  ${3}
<![endif]-->
```

### [hc.end] end tag

```html
<!-- // ${1:div} -->
```
